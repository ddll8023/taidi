## 全局速查（最高频规则）

1. 禁止 Element Plus：项目已全面迁移到自定义组件
2. Composition API：使用 `<script setup>`，不用 Options API
3. 图标统一 FontAwesome：所有图标使用 FontAwesome 6
4. 样式优先 Tailwind：原子类 > `@apply` > 自定义 CSS
5. API 参数 snake_case：请求参数/请求体必须用 snake_case
6. 统一响应格式：`{code, message, data}`，code===0 为成功
7. 路由懒加载：`()` => import()` 动态导入
8. Props 对象语法：defineProps 使用对象语法
9. Emits kebab-case：事件名使用 kebab-case
10. 语义化标签：header/main/section/footer，层级不超过 5 层
