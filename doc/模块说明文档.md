# 模块说明文档

本文档描述项目各模块的职责、核心导出、依赖关系和数据流向，作为后续开发参考。

---

## 一、项目概览

本项目是一个**上市公司财报智能问数助手**，采用前后端分离架构：

- **后端**：Python / FastAPI，负责数据处理、LLM 调用、数据库与向量库交互
- **前端**：Vue 3 / Tailwind CSS，负责用户界面与交互

核心业务流程：上传财报 PDF → 结构化抽取入库 → 基于财报数据与知识库回答财务问题

---

## 二、后端分层架构

```
api（路由层）  →  services（业务逻辑层）  →  models（ORM 层）
                    ↕                           ↕
              schemas（数据校验）          db（数据库连接）
                    ↕
              constants（业务常量）
                    ↕
              utils（工具函数）
              core（配置管理）
```

**依赖方向**：api → services → models / schemas / constants / utils / core。各层单向依赖，禁止反向调用。

## 五、核心数据流

### 流程1：财报处理流水线

```
用户上传 PDF（UploadModal）
  → api/analysis_data.upload_archive_only（保存文件 + 建档）
  → api/analysis_data.submit_and_run_single_parse
    → services/analysis_data/parse.py（线程池）
      → services/financial_report.py（元数据解析）
      → services/structured_report_extraction.py（LLM 结构化抽取）
      → services/analysis_data/helpers.py（字段清洗）
      → 入库到四张事实表
      → services/validation_log.py（各阶段校验记录）
```

### 流程2：智能问数（Chat）

```
用户发送消息（ChatView）
  → api/chat.py
    → services/chat/message.py（主流程调度）
      → 意图解析（LLM → IntentResult）
      → 插槽检查 & 澄清对话
      → services/chat/sql_builder.py（SQL 生成）
      → services/chat/executor.py（SQL 安全校验 + 执行）
      → services/visualization.py（图表生成）
      → 构建回答文本
```

### 流程3：知识库构建

```
系统初始化（SystemInitModal / BuildKnowledgeBaseModal）
  → services/knowledge_base/document.py（文档注册 + PDF 提取）
  → services/knowledge_base/chunk.py（文本切块）
  → services/knowledge_base/vectorize.py（Embedding → Milvus 写入）
  → services/knowledge_base/search.py（语义检索供 task3 使用）
```

### 流程4：任务三（深度研究）

```
用户导入附件6（Task3WorkbenchView）
  → services/task3/importer.py（题目提取 + 工作台初始化）
  → services/task3/runner.py（逐题执行）
    → services/task3/planner.py（LLM 规划：拆解为多步骤执行计划）
    → services/task3/executor.py（按步骤类型分发执行）
      → SQL_QUERY：调用 chat 模块
      → RETRIEVE_EVIDENCE：调用 knowledge_base/search
      → DERIVE_METRIC / AGGREGATE / COMPOSE_ANSWER
    → services/task3/verifier.py（多维度校验）
  → services/task3/exporter.py（导出 Excel）
```
