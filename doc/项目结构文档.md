# 项目结构文档

本文档仅说明项目自身的前后端文件结构与功能，不展开 `node_modules`、`dist`、`backend/result`、`backend/uploads`、`.venv` 等依赖、产物或运行时目录。

---

## 一、后端结构（backend）

### 1. backend 根目录

- `backend/.python-version`：声明后端开发环境使用的 Python 版本。
- `backend/pyproject.toml`：后端项目基础配置文件，定义项目元信息与 Python 版本要求。
- `backend/requirements.txt`：后端依赖清单。
- `backend/export_milvus.py`：Milvus knowledge_chunk_embedding 集合全量数据 JSON 备份导出脚本。
- `backend/uv.lock`：`uv` 生成的依赖锁定文件，用于固定依赖版本。

### 2. backend/app

- `backend/app/main.py`：FastAPI 应用入口，负责注册路由、初始化应用配置。

---

### 3. backend/app/api

- `backend/app/api/__init__.py`：API 包初始化文件。
- `backend/app/api/analysis_data.py`：财报上传、解析、查询、删除及附件数据导入接口。
- `backend/app/api/chat.py`：智能问数会话、消息、导出及图表访问接口。
- `backend/app/api/knowledge_base.py`：知识库文档、切块、向量化、状态查询相关接口。
- `backend/app/api/task2.py`：任务二工作台的导入、答题、批量处理、导出接口。
- `backend/app/api/task3.py`：任务三工作台的导入、规划、执行、验证、导出接口。

---

### 4. backend/app/config

#### 4.1 结构配置
- `backend/app/config/struct.yaml`：财务报表结构化抽取规则与字段配置。

#### 4.2 提示词配置
- `backend/app/config/prompts/chat.yaml`：智能问数相关提示词配置。
- `backend/app/config/prompts/derived_metrics.yaml`：派生指标、比率指标等提示词配置。
- `backend/app/config/prompts/task3.yaml`：任务三规划、执行、生成答案相关提示词配置。

---

### 5. backend/app/core

- `backend/app/core/config.py`：项目核心配置读取与统一管理。
- `backend/app/core/prompt_config.py`：提示词配置加载与访问封装。

---

### 6. backend/app/db

- `backend/app/db/database.py`：数据库引擎、会话工厂与数据库依赖定义。
- `backend/app/db/init_data.py`：数据库初始化逻辑。
- `backend/app/db/milvus.py`：Milvus 向量库连接与集合管理。

---

### 8. backend/app/schemas

- `backend/app/schemas/__init__.py`：Schema 包初始化文件。

- `backend/app/schemas/task3.py`：任务三规划、执行、引用、工作台相关模型定义。

### 9. backend/app/services

- `backend/app/services/company_basic_info.py`：公司基础信息导入与写入逻辑。
- `backend/app/services/financial_report.py`：财报元信息解析与财报主数据处理逻辑。
- `backend/app/services/fujian5_data_processor.py`：附件 5 研报类数据处理逻辑。
- `backend/app/services/structured_report_extraction.py`：财报结构化抽取主流程实现。
- `backend/app/services/validation_log.py`：校验日志写入与状态记录逻辑。
- `backend/app/services/visualization.py`：查询结果图表生成逻辑。

#### 9.1 backend/app/services/analysis_data
- `backend/app/services/analysis_data/__init__.py`：财报分析数据服务包初始化文件。
- `backend/app/services/analysis_data/delete.py`：财报及关联数据删除逻辑。
- `backend/app/services/analysis_data/normalize.py`：结构化抽取结果标准化与字段清洗逻辑。
- `backend/app/services/analysis_data/parse.py`：财报解析执行逻辑。
- `backend/app/services/analysis_data/persist.py`：结构化结果入库逻辑。
- `backend/app/services/analysis_data/query.py`：财报列表、详情、JSON 数据查询逻辑。
- `backend/app/services/analysis_data/upload.py`：财报文件上传与建档逻辑。

#### 9.2 backend/app/services/task2
- `backend/app/services/task2/__init__.py`：任务二服务子包初始化文件。
- `backend/app/services/task2/export.py`：任务二结果导出逻辑。
- `backend/app/services/task2/runner.py`：任务二单题/批量答题执行逻辑。
- `backend/app/services/task2/workspace.py`：任务二工作台导入、加载与状态维护逻辑。

#### 9.3 backend/app/services/chat
- `backend/app/services/chat/__init__.py`：智能问数对话服务包初始化文件。
- `backend/app/services/chat/message.py`：对话主流程：消息处理、意图解析、插槽检查、澄清、回答构建。
- `backend/app/services/chat/session.py`：会话管理：增删改查、关闭、重命名、导出。
- `backend/app/services/chat/sql_builder.py`：SQL 生成与解析服务。
- `backend/app/services/chat/executor.py`：SQL 执行与结果后处理服务。
- `backend/app/services/chat/query_handler.py`：特殊查询处理：跨表 TopN、多指标交集等。
- `backend/app/services/chat/context.py`：上下文消解、LLM 调用与通用工具服务。
- `backend/app/services/chat/metric.py`：指标处理与派生指标计算服务。

#### 9.4 backend/app/services/knowledge_base
- `backend/app/services/knowledge_base/__init__.py`：知识库文档管理服务包初始化文件。
- `backend/app/services/knowledge_base/document.py`：知识库文档注册、PDF 提取与文档列表查询服务。
- `backend/app/services/knowledge_base/chunk.py`：知识库文档分块与分块任务提交服务。
- `backend/app/services/knowledge_base/vectorize.py`：知识库向量化处理、向量化任务提交与进度查询服务。
- `backend/app/services/knowledge_base/search.py`：知识库语义检索与证据格式化服务。
- `backend/app/services/knowledge_base/metadata.py`：知识库元数据管理、缓存与统计服务。

#### 9.5 backend/app/services/task3
- `backend/app/services/task3/__init__.py`：任务三服务子包初始化文件。
- `backend/app/services/task3/executor.py`：任务三计划步骤执行逻辑。
- `backend/app/services/task3/exporter.py`：任务三结果导出逻辑。
- `backend/app/services/task3/importer.py`：任务三题目导入与初始化逻辑。
- `backend/app/services/task3/planner.py`：任务三问题拆解与执行规划逻辑。
- `backend/app/services/task3/runner.py`：任务三整体运行与批量处理逻辑。
- `backend/app/services/task3/verifier.py`：任务三执行结果校验逻辑。

- `frontend/index.html`：Vite 应用入口 HTML。
